# UniUsingCyberDataScienceInCybersecurity
Project Description: This project aims to reproduce and critically evaluate the results of the paper 
                      Context-Aware Phishing Email Detection Using Machine Learning and NLP. Throughout the project we will discuss various decisons made
                      and also perform some original experiments to try and understand the model and its capabilities.
Link to the original paper: http://arxiv.org/pdf/2603.27326
Link to the original github repository: https://github.com/AmitabhCh822/Phishing-Backend
Execution Instructions: The Jupyter notebook in the github repository is fully executable
Dataset Source: The dataset is made up of 3 different datasets, 2 of them are taken from kaggle, while the third dataset set is the enron dataset taken from zenodo
